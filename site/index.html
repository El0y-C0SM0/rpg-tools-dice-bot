RPG Tools - Dice Bot

Adicionar ao Discord

Apresentação

RPG Tools - Dice Bot é um bot de rolagem de dados para auxiliar jogadores de rpg. Ele possui a função de rolagem de dados padrão, com 10 tipos diferentes de dados, o bot também realiza rolagem secretas que somente quem solicitou a rolagem vai receber o resultado e tem uma função calculador que realiza caluculos básicos entre dois números para ajudar aqueles jogadores que tem uma certa dificuldade com matemática.

Comandos

Solicitar dados

Para solicitar dados, digite /roll e escreva o dado que deseja, colocando um número antes do d para pedir mais de um dado do mesmo tipo, colocando o valor do mod logo do dado indicando a operação e dando um espaço entre cada dado. Exemplo de solicitaçoes. O dado padrão do /roll é um d20. Para rolar um dado sem que os outro menbros vejam o valor use o comando /roll-master com as mesmas regras. Os seguintes dados são suportados: d2, d3, d4, d6, d8, d10, d12, d20, d100, d1000.

/roll 2d6 d20+4

/roll-master 2d12 d8-1

Calcular

É possivel calcular operações básicas como soma, subtração, divisão, potência e porcentagem. Use o camando /calc e digite o primeiro numero, o operador, o segundo numero e envie a mensagem
